# 명조 에코 부옵션 계산기

서버와 DB 없이 동작하는 정적 웹 계산기입니다. 브라우저에서 5개 에코의 치명타 확률, 치명타 피해, 코스트를 입력하면 세팅 총점과 개별 에코 등급을 계산합니다.

## 기능

- 1번 에코 4코스트 고정
- 총 코스트 12 제한
- 치명타 확률, 치명타 피해 부옵션 선택
- 세팅 총점 계산: `치명타 확률 합계 x 2 + 치명타 피해 합계 + 44`
- 세팅 등급과 개별 에코 등급 표시
- 브라우저 `localStorage` 자동 임시 저장
- CSV 내보내기/가져오기
- JSON 백업/복원
- 로그인, 저장 서버, PostgreSQL, Spring Boot 의존성 없음

## 실행

별도 빌드 과정이 없습니다. `index.html`을 브라우저에서 열면 됩니다.

## 배포

GitHub Pages, Cloudflare Pages, Netlify, Vercel 같은 정적 호스팅에 저장소 루트를 그대로 배포하면 됩니다.

GitHub Pages를 사용할 경우:

1. GitHub 저장소의 `Settings > Pages`로 이동
2. `Deploy from a branch` 선택
3. 배포할 브랜치와 `/ (root)` 선택
4. 저장 후 Pages URL에서 확인

## 파일 백업

CSV는 엑셀/구글시트 공유용이며, JSON은 계산기 상태를 정확히 복원하기 위한 백업용입니다. CSV 컬럼은 `version` 값을 포함해 향후 형식 변경에 대응할 수 있게 했습니다.
